# Sigil System  
_Version 1.0_

## Purpose

This document defines the sigil system —  
a modular symbolic language that encodes the Everlight’s mythic and structural meaning.

---

# Structure of a Sigil

A sigil consists of:

1. **Core Ring**  
   Represents the Everlight’s central harmony.

2. **Radial Lines**  
   Represent rotational energy and cosmic flow.

3. **Nodes**  
   Represent characters or forces:
   - Z = harmonic tri‑node  
   - Pillars = four slow nodes  
   - Tinies = cluster of tiny marks  

4. **Modifiers**  
   Represent emotional or narrative states:
   - chaos swirls  
   - calm arcs  
   - fire glyphs  
   - snack icons
