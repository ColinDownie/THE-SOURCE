# Rotation Logic  
_Version 1.0_

## Purpose

This document defines the logic that governs the Everlight’s rotational behaviour — the physical expression of its mythic calm.

---

# Core Principles

### 1. Smoothness  
No sudden changes.  
No jitter.  
No acceleration spikes.

### 2. Stability  
Rotation must remain consistent regardless of:
- temperature  
- load  
- long‑term wear  

### 3. Harmony  
Rotation speed subtly echoes:
- Z’s harmonic tone  
- the soundscape’s emotional arc  

---

# Control Logic

### Startup Sequence
1. soft ramp‑up  
2. stabilisation phase  
3. steady‑state rotation  

### Runtime Behaviour
- micro‑adjustments to maintain smoothness  
- optional slow breathing effect  
- no reactive changes to visitor movement  

### Shutdown Sequence
- gentle deceleration  
- soft stop  
- LED fade‑out  

---

# Summary

The rotation logic ensures the Everlight feels alive, calm, and deeply intentional.
