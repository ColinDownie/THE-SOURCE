# Visitor Flow Map  
_Version 1.0_

## Purpose

This document outlines how visitors move through the exhibit space to ensure clarity, immersion, and safety.

---

# Flow Overview

1. **Entry Zone**  
   - wall text  
   - sculpture revealed  
   - headphone pickup  

2. **Primary Viewing Zone**  
   - central position facing the Everlight  
   - optimal audio immersion  
   - space for groups  

3. **Reflection Zone**  
   - seating or standing area  
   - post‑experience contemplation  
   - optional secondary signage  

4. **Exit Zone**  
   - headphone return  
   - merchandise or catalogue area (optional)  

---

# Accessibility

- wide pathways  
- wheelchair‑friendly viewing angles  
- adjustable headphone height  
- clear signage  

---

# Summary

The visitor flow ensures a smooth, intuitive journey that supports the emotional arc of the soundscape.
