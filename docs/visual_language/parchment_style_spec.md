# Parchment Style Specification  
_Version 1.0_

## Purpose

This document defines the parchment‑style aesthetic used for lore pages, sigils, and mythic diagrams within **THE SOURCE**.

---

# Material Aesthetic

### Texture
- warm, fibrous, slightly uneven  
- subtle grain  
- faint imperfections  

### Colour Palette
- warm cream base  
- soft amber shadows  
- muted charcoal ink  
- occasional gold accents  

### Edge Treatment
- gently irregular  
- soft feathering  
- no harsh borders  

---

# Typography

### Primary Typeface
- serif  
- elegant  
- slightly calligraphic  

### Secondary Typeface
- monoline annotations  
- engineering notes  
- Tiny scribbles  

### Hierarchy
- large mythic titles  
- medium structural labels  
- tiny handwritten notes  

---

# Iconography

### Mythic Icons
- rings  
- spirals  
- radiant lines  

### Structural Icons
- rods  
- bearings  
- rotation arcs  

### Tiny Icons
- snacks  
- stamps  
- warning triangles  

---

# Usage

- lore pages  
- sigil diagrams  
- exhibit signage  
- worldbuilding inserts  

---

# Summary

The parchment style blends mythic elegance with playful detail —  
a visual bridge between Z’s cosmic calm and the Tinies’ chaotic scribbles.
